# 宇宙学笔记
## 教材推荐
1.《Physical Foundations of Cosmology 》by Viatcheslav Mukhanov （理论所的皮石老师翻译了这本书，翻译的相当好，这本也是笔记的主要参考教材.）  

2.Daniel Baumann 和 David Tong的《Cosmology》 （剑桥的两本宇宙学教材，后者更为基础一些.）  

3.《Cosmology》 by Steven Weinberg（除了看不懂没什么缺点）
